## Installation

```bash
hugo new site blog
cd blog
git clone https://wolffyhtl.github.io/Hugo-Landscape-Next themes/Hugo-Landscape-Next
```

Configure your own site based on exampleSite.

## Develop

```bash
git clone https://wolffyhtl.github.io/Hugo-Landscape-Next

just
```
